Discuss the scope of var, let, and const.

Answer:
The scope of the variable var, let and const show the different terms in declaration,hositing, block level and globally.

Var:
  1. 1.Var variable can declare with the function level and globally scope.
  2. 2.Var variable can allow to reassign the variable.
  3. 3.Var variable are allowed to hoisting the variable.

Let:
  1. 1.let variable can declare with the blcok level and globally scope.
  2. 2.Let variable can allow to reassign the variable.
  3. 3.Let variable are not allowed to hositing so that they can only accessible after the point of decaleration.

Const:
  1. 1.Const variable can declare with the block level and globally like let variable.
  2. 2.const variable are not allow to reassign the value after the declaration.
  3. 3.Like let variable, const variable are also not allowed to hoisting.

What do you mean by REST API?

Answer:
The term "REST API" refers to an application programming interface that complies with the design guidelines of the REST, or representational state transfer, architectural style.It outlines a collection of restrictions to be used while developing web services.The most frequent use of REST APIs is to send XML or JSON static resource representations.In the beginning, REST was developed as a set of rules to control communication on a complicated network like the internet. We can offer high-performing and dependable communication at scale using REST-based architecture. We can easily implement and modify it, bringing visibility and cross-platform portability to any API system.
